# Review Analysis Agent 📊

## Описание

MVP агента для анализа отзывов клиентов с использованием Claude API. Агент определяет тональность отзывов (позитивная/негативная/нейтральная), выявляет главные проблемы и темы, упоминаемые в отзывах, и предоставляет структурированный анализ с конкретными рекомендациями.

## Требования

- **Python**: 3.8 или выше
- **Зависимости**:
  - Flask 3.0.3 — веб-фреймворк для API
  - Anthropic 0.28.0 — SDK для Claude API
  - Python-dotenv 1.0.1 — загрузка переменных окружения
  - Pandas 2.0.3 — обработка CSV файлов
  - Pytest 7.4.0 — тестирование (опционально)

## Установка

### 1. Клонируйте репозиторий
```bash
git clone <repository-url>
cd review_agent
```

### 2. Установите зависимости
```bash
pip install -r requirements.txt
```

### 3. Настройте переменные окружения
```bash
cp .env.example .env
```

### 4. Добавьте ваш API ключ
Отредактируйте файл `.env` и установите ваш ANTHROPIC_API_KEY:
```
ANTHROPIC_API_KEY=sk-ant-xxxxxxxxxxxxx
```

## Запуск

```bash
python main.py
```

API запустится на **http://localhost:5000**

## API Примеры

### 1. Анализ отзывов через JSON

**POST /analyze**

```bash
curl -X POST http://localhost:5000/analyze \
  -H "Content-Type: application/json" \
  -d '{
    "reviews": [
      "Отличный сервис! Быстрая доставка и качественный товар.",
      "Разочарован. Товар не соответствует описанию. Плохое качество упаковки.",
      "Нормально. Ничего особенного, но и не плохо. Цена адекватная."
    ]
  }'
```

### 2. Анализ отзывов через CSV файл

**POST /analyze** (multipart/form-data)

```bash
curl -X POST http://localhost:5000/analyze \
  -F "file=@reviews.csv"
```

**Формат CSV файла:**
```
review
"Отличный сервис! Быстрая доставка и качественный товар."
"Разочарован. Товар не соответствует описанию."
"Нормально. Цена адекватная."
```

### 3. Проверка здоровья сервиса

**GET /health**

```bash
curl http://localhost:5000/health
```

Ответ:
```json
{
  "status": "healthy"
}
```

## Топ-3 проблемы и решения

### 1. Timeout при анализе большого количества отзывов

**Проблема**: Анализ 1000 отзывов может превышать лимит в 30 секунд.

**Решение**:
- Разбейте отзывы на батчи по 200-300 отзывов
- Отправляйте несколько запросов параллельно
- Увеличьте значение `CLAUDE_API_TIMEOUT` в `analyzer.py` (осторожно!)

```python
# В analyzer.py
CLAUDE_API_TIMEOUT = 60  # увеличить с 30 до 60 секунд
```

### 2. Низкое качество анализа для коротких отзывов

**Проблема**: Отзывы короче 20 символов могут содержать недостаточно информации для анализа.

**Решение**:
- Используйте минимальную длину 20+ символов вместо 15
- Проверьте исходные отзывы на релевантность
- Убедитесь, что отзывы содержат достаточно контекста

### 3. Неправильный парсинг CSV с кириллицей

**Проблема**: CSV файлы с русским текстом могут неправильно парсится.

**Решение**:
- Убедитесь, что файл сохранён в кодировке UTF-8
- В Excel: File → Save As → CSV UTF-8
- В Python: используйте правильную кодировку при создании файла

```python
import pandas as pd
df = pd.read_csv('reviews.csv', encoding='utf-8')
```
